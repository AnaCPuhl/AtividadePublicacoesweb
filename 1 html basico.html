O que é uma publicação web

Publicação web é o ato de criar e disponibilizar conteúdo online para que qualquer pessoa com acesso à internet possa ver. É como se você estivesse publicando um livro, revista ou jornal, mas em vez de papel, você usa um site ou blog.

Existem dois tipos principais de aplicações web:

Aplicações Web Estática

  • Conteúdo fixo, não alterado dinamicamente.
  • Exemplos: sites de apresentação, portfólios.
  • Vantagem: Fácil de criar e manter.
  • Desvantagem: Pouca interação com o usuário.
