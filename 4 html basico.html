O que é Servidor web

Software que responde a requisições HTTP, enviando arquivos (HTML, CSS, JS) para o navegador.

  • Principais Servidores: Apache e Nginx.
  • Backup: Cópia de segurança dos dados
  • Restauração: Reposição dos dados a partir do backup.
